## Comandos para poder usar la cosa esta

Dependencias:
```npm install```

Para prisma:
```npm prisma generate```

Compilado:
```npm run build```

Ejecutar con node:
```node dist/index.js```

Compilar y ejecutar al mismo tiempo:
```npm run build && node dist/index.js```
