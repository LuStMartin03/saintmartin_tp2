## Comandos utililes

Dependencias:
```npm install```

Para prisma:
```npx prisma generate```
```npx prisma db push --force-reset```

Compilado:
```npm run build```

Ejecutar con node:
```node dist/index.js```

Compilar y ejecutar al mismo tiempo:
```npm run build && node dist/index.js```
